# ChatMine

桌面端 AI 聊天软件，支持本地运行、微信聊天记录挖掘、AI 人格复刻。

## 功能

- **AI 对话**：本地桌面端 AI 聊天，多实例并行
- **聊天挖掘**：解析微信聊天记录，提取对话模式和个人风格
- **人格复刻**：基于聊天数据训练 / 模拟特定人格的 AI 回复

## 系统要求

- Windows 10/11
- Python 3.10+

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
python app.py
```

## 构建

```bash
python build_exe.bat
```
